# Customer Churn Risk Intelligence System

Predict high-risk churn customers and target top 10% users for retention.

## Features
- Logistic Regression Model
- Risk Ranking Engine
- Top 10% Capture Optimization
- Cumulative Gain Curve
- Retention ROI Simulation
- A/B Testing Framework

## Business Objective
Top 10% highest-risk customers must capture at least 40% of churn.

## Tech Stack
- Python
- Scikit-learn
- FastAPI
- Streamlit / React
- Recharts

## Run Backend
uvicorn main:app --reload

## Run Frontend
streamlit run app.py
